# AI Fluency: Framework and Foundations

A personal learning project exploring the core concepts, frameworks, and foundations of AI — with a focus on large language models, prompt engineering, and practical AI literacy.

## Structure

```
├── 01_foundations/       # Core AI/ML concepts and terminology
├── 02_llm_basics/        # How LLMs work — transformers, tokens, context
├── 03_prompt_engineering/ # Prompting techniques and patterns
├── 04_frameworks/        # AI frameworks, tools, and ecosystems
├── 05_practical/         # Hands-on exercises and experiments
└── notes/                # Freeform notes and reading summaries
```

## Goals

- Build a solid mental model of how modern AI systems work
- Understand prompt design and its impact on model outputs
- Explore real-world AI use cases and limitations
- Document learnings in a structured, reusable way
